IsIntersected_Point_Polygon

    Arguments:
        1) real: point - x
        2) real: point - y
        3) (real[2][]): polygon vertices
    Returns:
        bool: intersection

Returns true if the given point intersects with the polygon.

The polygon can be convex, concave, or complex (self-intersecting).
