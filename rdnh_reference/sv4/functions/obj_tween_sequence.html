ObjTweenSequence_Create

    Arguments:
        1) real?: sizeReserve
    Return Type:
        real

Creates a tween sequence object and returns its ID.

sizeReserve is the amount of memory to pre-allocate for the tween sequence's command list.
This argument may be omitted to use the default value.
