Introduction

In the following sections, arguments listed after the constant name like TWEEN_SOMETHING(arg1, arg2) refer to the arguments passed after the constant in the ObjRender_AddTween function.

Note: Argument values must not exceed 9,007,199,254,740,975 (2^53-17).

Variables

These are not actually commands. They can be used in the same places as value arguments, but behave as variables.

TWEEN_V0: variable at index 0

TWEEN_V1: variable at index 1

TWEEN_V2: variable at index 2

TWEEN_V3: variable at index 3

Sequence-related/General Purpose

TWEEN_WAIT(frames): delays execution of the next command by frames

TWEEN_PRINT(level = LOG_USER): prints debug info about the sequence

TWEEN_NOTIFY_EVENT(scriptId, eventType, args...): NotifyEvent

TWEEN_NOTIFY_EVENT_OWN(eventType, args...): NotifyEventOwn

TWEEN_NOTIFY_EVENT_ALL(eventType, args...): NotifyEventAll

Variables and Math

TWEEN_VSET(var&, value): var = value

TWEEN_VADD(var&, value): var += value

TWEEN_VSUB(var&, value): var -= value

TWEEN_VMUL(var&, value): var *= value

TWEEN_VDIV(var&, value): var /= value

TWEEN_VMOD(var&, value): var %= value

TWEEN_VSIN(var&, value): var = sin(value)

TWEEN_VCOS(var&, value): var = cos(value)

TWEEN_VRAND(var&, minValue, maxValue): var = prand(minValue, maxValue)

Jumps

All jumps use absolute offsets.

TWEEN_JUMP(jumpDest): unconditional jump to jumpDest

TWEEN_JUMP_DEC(varCounter&, jumpDest): decrement varCounter, if varCounter > 0: jump to jumpDest

TWEEN_JUMP_E(a, b, jumpDest): if a == b: jump to jumpDest

TWEEN_JUMP_NE(a, b, jumpDest): if a != b: jump to jumpDest

TWEEN_JUMP_L(a, b, jumpDest): if a < b: jump to jumpDest

TWEEN_JUMP_LE(a, b, jumpDest): if a <= b: jump to jumpDest

TWEEN_JUMP_G(a, b, jumpDest): if a > b: jump to jumpDest

TWEEN_JUMP_GE(a, b, jumpDest): if a >= b: jump to jumpDest
