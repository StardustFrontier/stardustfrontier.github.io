ObjRender_AddTween

    Arguments:
        1) real: objID
        2) real const: tweenType
        3+) real: args...
    Return Type:
        real

Adds a tween command to the object's tween sequence and returns its index.

Required arguments for each type vary, see the list of commands for more details.

ObjRender_RemoveTween

    Arguments:
        1) real: objID

Removes the last tween command from the object's tween sequence.

ObjRender_InitializeTweenSequence

    Arguments:
        1) real: objID
        2) real: size

Initializes the object's tween sequence, optionally reserving space (0 or NULL to ignore) for size commands.

ObjRender_SetTweenSequence

    Arguments:
        1) real: objID
        2) real: objSeqID

Sets the object's tween sequence to objSeqID, which must be an ObjTweenSequence.

ObjRender_CopyTweenSequence

    Arguments:
        1) real: objID
        2) real: objID2

Copies objID2's tween sequence to objID.

ObjRender_PlayTweenSequence

    Arguments:
        1) real: objID

Starts playing the object's tween sequence from the beginning.

ObjRender_StopTweenSequence

    Arguments:
        1) real: objID
        2) bool: bCancelTweens

Stops playing the object's tween sequence.

if bCancelTweens is true, cancels any CHANGE type tweens.

ObjRender_PauseTweenSequence

    Arguments:
        1) real: objID
        2) bool: bCancelTweens

Pauses the object's tween sequence without resetting its state.

if bCancelTweens is true, cancels any CHANGE type tweens.

ObjRender_ResumeTweenSequence

    Arguments:
        1) real: objID

Resumes playing a previously paused object's tween sequence.

ObjRender_PatchTweenJump

    Arguments:
        1) real: objID
        2) real: jumpIndex

Patches the jump at jumpIndex in the tween sequence to point to the next added tween command.

ObjRender_SetTweenSequenceOnFinish

    Arguments:
        1) real: objID
        2) function: funcOnFinish

Sets the function to call after the tween sequence finishes executing.

funcOnFinish cannot be task or sub and must take 1 parameter.

ObjRender_GetTweenSequenceTimer

    Arguments:
        1) real: objID
    Return Type:
        real

Returns the tween sequence's current timer value. This is set by TWEEN_WAIT and counts down to 0.

ObjRender_SetTweenSequenceTimer

    Arguments:
        1) real: objID
        2) real: timer

Sets the tween sequence's timer value directly. A tween of type TWEEN_WAIT should be used instead for most cases.

ObjRender_GetTweenSequenceIndex

    Arguments:
        1) real: objID
    Return Type:
        real

Returns the tween sequence's current instruction index.

This may be inaccurate during execution, since tween sequences run ahead of scripts.

ObjRender_SetTweenSequenceIndex

    Arguments:
        1) real: objID
        2) real: index

Sets the tween sequence's instruction index directly.

This can be useful for setting the initial index to start at.

ObjRender_TweenSequenceIsStopped

    Arguments:
        1) real: objID
    Return Type:
        bool

Returns true if the sequence is stopped or finished, otherwise returns false.
